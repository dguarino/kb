How to upload your tags

Before using this upload page, be sure you did the following:
  1. Export the list of article using the 'Export CSV' link above.
  2. Open the 'article.csv' file you get in the exported zip.
  3. The first line contains headers for the fields. Append there a new tag label (rememeber the separation comma!).
  4. Append at the end of each line the tag value (rememeber the separation comma!).
  5. Save and close the CSV file.

      6. Now you can upload your tags for articles ...
